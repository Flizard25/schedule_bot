# Schedule_bot
## О боте
Это телеграмм-бот, который скидывает расписание. Ссылка на бота в телеграмм: http://t.me/schedule_in_university_bot. Если вы хотите посмотреть код не скачивая файла, то нажмите на название файла
## Как скачать
1. Нажмите зелёную кнопку "Code" ЛКМ
2. Нажмите на "Download ZIP" ЛКМ
3. Подождите когда ZIP-файл скачается, зайдите в проводник и нажмите на ZIP ПКМ и выберите "Извлечь всё" ЛКМ
4. Появится окно, в котором нажмите на "Извлечь всё" ЛКМ
