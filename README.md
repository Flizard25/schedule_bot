# Schedule_bot
## О боте
Это телеграмм-бот, который скидывает расписание. Ссылка на бота в телеграмм: http://t.me/schedule_in_university_bot. Если вы хотите посмотреть код не скачивая файла, то нажмите на название файла
## Как скачать
1. Нажмите зелёную кнопку "Code" ЛКМ (обведена красным)
![alt text](Code_button.png "Title")
2. Нажмите на "Download ZIP" ЛКМ (также обведена красным)
![alt text](Downloadzip.png "Title")
3. Подождите когда ZIP-файл скачается, зайдите в проводник и нажмите на ZIP ПКМ и выберите "Извлечь всё" ЛКМ4. 
4. Появится окно, в котором нажмите "Извлечь" ЛКМ
